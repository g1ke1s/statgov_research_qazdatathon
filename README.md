# Kazakhstan Socio-Economic Datathon Project

This repository contains **Jupyter Notebooks** developed as part of the **Government Datathon** focused on analyzing socio-economic challenges in Kazakhstan using official household and labor statistics.

The project follows a full analytical pipeline:
- hypothesis formulation  
- data exploration and validation  
- statistical analysis  
- insights and policy-oriented conclusions  

---

## Data Sources

The analysis is based on official survey data provided for the datathon, including:
- Quality of life and life satisfaction indicators  
- Household income and expenditure surveys  
- Time-use statistics  
- Labor market and decent work indicators  

The rest information is available through the official datathon platform:

## Tools

- Python (Pandas, NumPy, SciPy)
- Statistical testing and exploratory data analysis (EDA)
- Visualization (Matplotlib, Seaborn)

---

## Project Goal

To identify **key socio-economic patterns and risks** affecting households in Kazakhstan and propose **data-driven insights** relevant for policy design and social development.
